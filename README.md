## Getting Started

- Download the .jar in latest release
- Run `java -jar karl-x.x.x.jar <file.karl>`

## Example

```js   
use std::io;

declare int: number = 150; // this is a comment
declare bool: boolean = true;
declare int: num = 1;

declare func main::(): int -> {
  return 1;
}

io:Show(main() + num);
// -> 2
```
