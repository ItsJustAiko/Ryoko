## Karl, an homemade programming language

## Example

```js
int: number = 150; // this is a comment
bool: boolean = true;
int: num = 1;

func main::(): int -> {
  return 1;
}

show(main() + num);
// -> 2
```

## How to use

- Download the .jar in latest release
- Run `java -jar karl-x.x.x.jar <file.karl>`
